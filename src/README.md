# src

React single-page portfolio source.

- `App.tsx`: page composition and role perspective state.
- `main.tsx`: Vite/React entrypoint.
- `components/`: presentational UI components.
- `data/`: local fallback career data and public content rules.
- `styles/`: global layout and visual system.

Google Sheets will be treated as the external read-only CMS. Local files under `data/` remain the fallback source and should not contain sensitive private information.
