# components

Reusable presentation components for the single-page portfolio.

- `Header.tsx`: compact in-page navigation.
- `Hero.tsx`: candidate identity, positioning, key tags, and primary actions.
- `ProfilePhoto.tsx`: stable profile photo or placeholder area.
- `ProjectCard.tsx`: project evidence, fit explanation, and case-study details.
- `EvidenceIndex.tsx`: shared evidence tag counts across projects and experiences.
- `ExperienceTimeline.tsx`: career timeline with function-based highlights.
- `SkillMatrix.tsx`: grouped technical and work-practice skills.
- `Section.tsx`: semantic page section wrapper.

Components should stay presentational. Career content belongs in `src/data` or the future Google Sheets CMS.
