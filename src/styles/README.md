# src/styles

전역 스타일을 책임 단위로 나눈다.

## Files

- `global.css`: CSS import 진입점.
- `tokens.css`: 색상, 배경, 간격, 최대 폭 토큰.
- `base.css`: reset, 기본 타이포그래피, focus-visible.
- `layout.css`: 전체 페이지, 첫 화면 dashboard, 문서 섹션, 푸터 레이아웃.
- `components.css`: 첫 화면, 문서 제목, 태그 등 공통 화면 요소 스타일.
- `document.css`: 연혁 행, 상세 패널, 자기소개, 대표 업무, 기술 맥락 스타일.
- `responsive.css`: 모바일과 print 대응.

## Rules

- 데스크톱 첫 화면에는 이름만 크게 보이지 않게 하고, 요약·최근 연혁·역량이 함께 보이게 한다.
- 연혁은 별도 축 컬럼에 직선을 두고 그 위에 점을 얹으며, `기간 · 소속 · 역할 · 요약`을 한 줄 흐름으로 읽히게 한다.
- 상세 내용은 기본 행에 과하게 넣지 않고 선택 패널에서 보여준다.
- 모바일에서는 한 컬럼 문서형 흐름으로 전환한다.
- 프린트에서는 배경과 장식을 제거하고 경력 정보가 읽히게 한다.
